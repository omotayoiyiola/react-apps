FETCH
fetch is a built-in JavaScript function used to make HTTP 
requests (like GET, POST, PUT, DELETE) from the browser to a 
server or API.

It’s part of the Fetch API, which allows you to asynchronously 
request resources and handle responses (like JSON, text,
HTML, etc.) — all using Promises.

Syntax
fetch(url, options)
  .then(response => response.json())
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error('Error:', error);
  });


  fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(res => res.json())
  .then(data => console.log(data));
